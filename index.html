플레이 경험과 시스템 구조를 함께 설계하는 게임 개발자 안병찬

Unity 기반 게임 개발을 중심으로, 전투 시스템, 성장 구조, UI, 데이터 관리, ECS 기반 시스템 설계를 직접 구현해 왔습니다. 기능을 단순히 붙이는 개발보다, 유지보수와 확장을 고려한 구조적인 개발을 지향합니다.

소개

강점

  • 전투, 성장, UI처럼 플레이어가 직접 체감하는 영역을 구현하는 데 강점이 있습니다.
  • 기능이 늘어나도 버틸 수 있도록 시스템 흐름과 책임을 나누는 구조 설계를 중요하게 생각합니다.
  • 데이터 분리, 리소스 관리, 리팩터링을 통해 프로젝트 유지보수성을 높이는 방향으로 작업합니다.

개발 성향

  • 문제를 재현하고 원인을 좁히는 방식으로 접근합니다.
  • 빠르게 동작하는 코드보다 나중에도 고치기 쉬운 구조를 선호합니다.
  • 플레이 감각과 시스템 구조를 따로 보지 않고 함께 맞춰 나갑니다.

프로젝트

주요 프로젝트

Slime Defense

Unity ECS(DOTS) 기반으로 제작 중인 모바일 디펜스 프로젝트입니다. 전투, 경험치 획득, 레벨업 3지선다, 강화 반복으로 이어지는 짧고 밀도 높은 플레이 루프를 설계했습니다.

  • ECS 시스템 그룹 기반 업데이트 흐름 구성
  • 레벨업 선택지와 강화 수치의 데이터 중심 설계
  • 터치 입력, 조준 UI, 모바일 플레이 감각 조정
  • Addressables 기반 리소스 관리 구조 정리

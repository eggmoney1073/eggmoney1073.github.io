핵심 키워드

Unity C# ECS / DOTS Addressables UI 시스템 모바일 게임 구조 게임플레이 설계 리팩터링

개발 성향

구조 중심 시스템 흐름과 책임 분리를 먼저 정리합니다.
문제 해결 막히는 지점을 재현하고 원인을 좁혀 해결합니다.
실전 지향 게임 루프와 플레이 감각이 살아 있는 구현을 선호합니다.
확장 고려 추가 기능이 붙어도 버티는 구조를 목표로 합니다.

소개

전투, 성장, 조작, UI와 같은 플레이어 체감 영역을 직접 구현하면서, 그 기반이 되는 시스템 구조까지 함께 다듬는 개발자입니다. 최근에는 Unity ECS(DOTS) 기반 구조와 Addressables를 활용한 리소스 관리, 모바일 환경을 고려한 게임 흐름 설계에 집중하고 있습니다.

강점

  • 전투 루프, 레벨업 선택지, UI 피드백 등 플레이 감각에 직접 연결되는 요소 구현
  • 기능 추가에 흔들리지 않도록 시스템 책임을 나누고 흐름을 재정리하는 구조 설계
  • Unity Addressables, ScriptableObject, 데이터 분리 구조를 통한 관리 편의성 확보
  • ECS 환경에서 스폰, 이동, 판정, 계산, 상태 처리 흐름을 단계별로 구성하는 설계 경험

지향점

  • 한 번 동작하는 코드보다, 이후 수정과 확장이 가능한 구조를 만드는 개발
  • 플레이어가 체감하는 재미를 중심으로 시스템과 데이터를 설계하는 개발
  • 장기적으로는 한국어 중심의 컴퓨팅 환경과 목적 특화형 소프트웨어까지 확장 가능한 개발 역량 확보

프로젝트

직접 설계하고 구현한 프로젝트 중심으로, 장르적 목표와 기술적 포인트를 함께 정리했습니다.

Slime Defense

Unity ECS(DOTS) 기반 모바일 디펜스 / 서바이벌 성장형 프로젝트

주요 프로젝트

개요

화면 중앙에 고정된 플레이어가 터치 조작으로 조준 방향을 제어하고, 몰려오는 적을 처치하며 성장하는 구조의 프로젝트입니다. 전투, 경험치 획득, 레벨업 3지선다, 강화 반복이라는 핵심 루프를 ECS 기반 흐름으로 구성했습니다.

기술 포인트

  • 스폰, 이동, 판정, 계산, 관리 시스템을 그룹 단위로 분리해 업데이트 흐름을 명확히 설계
  • 레벨업 선택지, 무기 강화, 희귀도 구조를 데이터 중심으로 설계
  • 모바일 조작을 고려한 터치 입력, 조준 UI, 화면 내 상호작용 구조 구현
  • Addressables 기반 UI 및 리소스 로딩 구조를 적용해 직접 경로 하드코딩을 줄이는 방향으로 정리

Monster World

3D 액션 RPG 프로젝트

게임플레이 중심

개요

다양한 무기와 전투를 중심으로 한 3D 액션 RPG 프로젝트입니다. 플레이어 조작, 조준, 공격, 경험치 성장, 몬스터 포획 및 재소환 등 전투와 운용의 재미를 함께 담는 방향으로 개발했습니다.

기술 포인트

  • 플레이어 이동, 조준, 무기 전환, 원거리 공격 흐름 구현
  • 순찰, 추적, 공격, 복귀를 포함한 상태머신 기반 적 AI 구성
  • 경험치 획득과 레벨업에 따른 성장 시스템 구현
  • 몬스터 포획, 저장, 재소환 및 슬롯 기반 저장 구조 설계
Unity C# 3D 전투 상태머신 AI 저장 시스템

기술 스택

주력 기술은 Unity와 C#이며, 프로젝트 성격에 따라 ECS, Addressables, 데이터 분리 구조를 적용하고 있습니다.

Core

Unity C# 게임플레이 구현 UI 시스템

Architecture

ECS / DOTS ScriptableObject 데이터 중심 설계 시스템 그룹 분리

Pipeline

Addressables Git / GitHub 리소스 관리 디버깅 / 리팩터링

개발 방식

구현 과정에서는 문제 재현, 원인 분리, 구조 개선의 순서로 접근합니다. 기능을 빠르게 붙이는 것보다 이후 수정이 쉬운 형태를 선호합니다.

1. 문제를 분명하게 정의

조작감, 성능, 구조 꼬임, 데이터 관리 문제를 증상 단위로 나누고, 어디서부터 잘못되는지 좁혀 갑니다.

2. 흐름과 책임을 정리

시스템 간 순서, 데이터 소유 위치, UI와 게임 로직의 역할을 나눠 유지보수하기 쉬운 형태로 바꿉니다.

3. 플레이 감각까지 확인

기능이 동작하는 것만으로 끝내지 않고, 전투 템포와 성장 체감처럼 실제 플레이에서 중요한 부분까지 점검합니다.

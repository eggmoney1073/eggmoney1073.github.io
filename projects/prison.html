Project Detail

Prison Resource Tycoon

AI를 활용해 게임 시스템을 설계하고 구현까지 완성한 프로젝트입니다. 단순 구현이 아닌 요구사항 정의, 구조 설계, 문제 해결 과정에 집중하여 개발했습니다.

Overview

프로젝트 개요

개발 방식

AI 협업 개발

요구사항 정의와 구조 설계를 중심으로 AI를 활용해 구현

핵심 구조

자동화 루프

수동 → 자동화로 확장되는 게임 구조 설계

플레이 흐름

상태 기반 구조

Start → Guide → Play → End 상태 설계

개발 포인트

문제 해결 중심

구현보다 디버깅과 개선 과정에 집중

핵심 게임 루프

자원 채굴 → 가공 → 납품 → 보상 → 업그레이드 → 자동화 구조를 기반으로 플레이가 진행되며, 플레이어의 행동이 점차 NPC로 대체되는 자동화 흐름을 설계했습니다.

Process

개발 과정

1. 시스템 설계
게임 전체 흐름과 자원 이동 구조, NPC 역할을 먼저 정의
2. 요구사항 세분화
채굴, 운송, 생산, 납품, UI 동작까지 상세 조건을 구체화
3. AI 협업 구현
설계한 구조를 기반으로 단계적으로 기능 구현 요청
4. 반복 개선
문제 발생 시 원인을 정의하고 구조 단위로 수정

Insight

핵심 역량

요구사항 정의 능력

단순 기능 요청이 아닌 시스템 단위로 요구사항을 정의하고, 게임의 전체 흐름을 먼저 설계한 뒤 구현을 진행했습니다.

문제 해결 능력

UI 가려짐, 이벤트 타이밍 문제, NPC 동작 오류 등 실제 개발 과정에서 발생한 문제를 분석하고 해결했습니다.

AI 활용 능력

AI를 단순 코드 생성 도구로 사용하지 않고, 반복적인 피드백을 통해 결과를 개선하는 방식으로 활용했습니다.

시스템 설계 능력

플레이어 → NPC → 자동화로 확장되는 구조를 설계하며 확장성과 유지보수를 고려한 시스템을 구성했습니다.

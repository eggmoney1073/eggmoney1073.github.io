PROJECT 02

Monster World

몬스터 포획, 전투, 성장, 저장 구조를 갖춘 3D 액션 RPG 프로젝트입니다. 단순 전투 구현이 아니라, 몬스터 재사용 구조와 데이터 관리, 상태 제어까지 포함해 시스템 중심으로 설계했습니다.

Unity C# State Machine Object Pooling JSON Data

프로젝트 개요

플레이어는 몬스터와 전투하고, 포획한 몬스터를 저장 및 소환하여 함께 전투할 수 있습니다. 전투 경험치, 성장, 도감, 저장 슬롯까지 연결되는 구조를 하나의 게임 흐름으로 구현했습니다.

핵심 기능

  • 몬스터 포획 및 소환 시스템
  • 상태 전환 기반 몬스터 AI
  • 스킬 사용 구조와 오브젝트 재사용
  • JSON 기반 데이터 로드 및 관리
  • Shader Graph를 활용한 연출 제작

State Machine 기반 AI

몬스터의 행동을 상태 단위로 분리해 순찰, 추적, 전투, 복귀 흐름을 제어했습니다. 아군과 적군이 같은 오브젝트 구조를 공유하면서도 상황에 따라 다른 행동을 하도록 설계했습니다.

Object Pooling 최적화

몬스터, 스킬, 투사체, 이펙트 등을 풀링 구조로 관리해 런타임에서의 생성과 삭제 비용을 줄이고 보다 안정적인 플레이 환경을 목표로 했습니다.

데이터 기반 개발

Excel 데이터를 JSON으로 변환해 게임 수치와 테이블을 관리했습니다. 하드코딩을 줄이고, 밸런스 조정과 데이터 수정이 편한 구조를 만드는 데 집중했습니다.

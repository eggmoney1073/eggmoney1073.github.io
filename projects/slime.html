PROJECT 01

Slime Defense

Unity ECS 기반으로 제작한 모바일 디펜스 프로젝트입니다. 플레이어를 중앙에 두고, 몰려오는 적을 상대하며 성장하는 짧고 밀도 높은 전투 루프를 목표로 설계했습니다.

Unity ECS C# Addressables Mobile Defense + Survival

프로젝트 개요

디펜스 장르에 서바이벌 성장 구조를 결합한 모바일 프로젝트입니다. 전투 중 경험치를 획득하고, 레벨업 시 3지선다 선택을 통해 전투 스타일을 강화하는 구조를 중심으로 만들었습니다.

핵심 포인트

  • 짧은 플레이 타임 안에서 빠르게 성장 체감을 주는 구조
  • 레벨업 선택지 기반의 전투 강화 루프
  • 모바일 환경에 맞춘 조작과 UI 흐름
  • ECS 기반의 구조적 확장성 고려

전투와 성장의 연결

적 처치와 경험치 획득, 레벨업 선택, 강화 결과가 즉시 다음 전투에 반영되도록 구성해 플레이 흐름이 끊기지 않도록 설계했습니다.

ECS 기반 설계

엔티티와 시스템 중심으로 구조를 분리해 데이터 흐름이 명확하도록 구성했고, 이후 기능 확장과 최적화에 대응하기 쉬운 형태를 지향했습니다.

모바일 플레이 감각

짧은 세션에서도 재미가 빠르게 전달되도록 게임 템포를 조정하고, 터치 기반 조작 흐름과 가시성을 고려해 UI와 플레이를 정리했습니다.
